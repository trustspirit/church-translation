## 2. 교회 지도자 이름 표기 원칙

- 기본 형식: `[이름] [중간 이름 한글 음가] [성] [직책]`
  - 예: 러셀 엠 넬슨 회장, 댈린 에이치 옥스 회장, 제프리 알 홀런드 장로
- 중간 이니셜은 한글 음가로 표기하고 마침표는 생략한다.
  - 권장: 러셀 엠 넬슨 / 피해야 함: 러셀 M. 넬슨, 러셀 엠. 넬슨, 러셀 M 넬슨
- 성만 나온 경우: 넬슨 회장, 홀런드 장로, 존슨 자매
- 같은 성을 가진 인물이 여럿 등장하거나 혼동 가능성이 있으면 전체 이름 사용
- 공식 한국어 표기가 확실하지 않은 인명은 임의로 음역하지 말고
  `[이름 공식 표기 확인 필요]`로 표시한다.
- **직책·호칭은 시점에 따라 달라질 수 있다.** 번역할 때는 항상 원문이
  부여한 직책(Elder / President / Sister 등)을 우선한다. 참고집 1장 표의
  직책 표기는 anchor 힌트일 뿐 절대 표기가 아니다.
  - 예: 같은 인물이 시점에 따라 장로 → 회장(정원회 회장 대리) →
    회장(제일회장단)으로 바뀔 수 있다. 원문 시점의 직책을 따른다.

## 3. 직책·호칭

| English | 한국어 |
| --- | --- |
| President (교회·스테이크·정원회·선교부·조직 회장) | 회장 |
| Elder | 장로 |
| Sister | 자매 |
| Brother | 형제 |

- 호칭은 이름 뒤에 붙인다. 예: 넬슨 회장, 홀런드 장로, 존슨 자매
- 선교사 호칭: Elder Smith → 스미스 장로 / Sister Johnson → 존슨 자매
- 두 명 이상: Elders Smith and Brown → 스미스 장로와 브라운 장로 /
  Sisters Kim and Lee → 김 자매와 이 자매
- 여성 지도자: Sister + 이름 → 보통 "자매"로 옮김.
  예: Sister Bonnie H. Cordon → 보니 에이치 코든 자매
- elder가 선교사 호칭이 아니라 "연장자" 또는 교회 직분을 뜻하는 경우는
  문맥에 따라 다르게 번역한다.

## 4. 경전 인물

교회 한국어 경전에서 사용하는 표기를 일반 외래어 표기보다 우선한다.
아래 표는 자주 등장하는 anchor 인물만 정리한 것이며, **표에 없거나
의심스러운 인물은 그 자리에서 임의 음역하지 말고 절차에 따라 검증한다.**

### 경전 인물 표기 검증 절차

1. churchofjesuschrist.org 한국어판 성구 안내(Guide to the Scriptures)에서
   확인. URL 패턴: `/study/scriptures/gs/{영문 이름 소문자}?lang=kor`
   (예: `gs/laman?lang=kor`, `gs/enoch?lang=kor`)
2. 안 나오면 한국어 경전 본문에서 해당 책·장·절을 검색.
3. 그래도 확인 불가능하면 임시 음역 + `[경전 인물 표기 확인 필요]` 표시.

## 5. 공식 표기가 없는 인명

### 한국인 이름

- 가능한 한 원래 한글 이름을 사용한다.
- 영어 자료에 로마자로만 표기되어 있고 확실한 한글 표기를 알 수 없으면
  임의로 단정하지 말고 `[한글 이름 확인 필요]`로 표시한다.
  - 예: Kim Young-ho → 김영호 `[한글 이름 확인 필요]`
- 성과 이름 구분이 불확실하거나 여러 한글 표기가 가능하면 마찬가지로 표시한다.

### 그 외 외국인 이름

- 원어 발음에 가까운 자연스러운 한글 음역을 사용한다.
- 같은 문서 안에서는 같은 인물을 항상 같은 방식으로 표기한다.
- 처음 등장할 때 원문 이름을 괄호로 병기할 수 있다.
  - 예: Sarah Williams → 세라 윌리엄스(Sarah Williams)

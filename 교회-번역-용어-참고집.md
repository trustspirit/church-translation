## 5. 문체·어체 원칙

### 어체 (격식 수준)

- **말씀·설교·기사 본문은 합쇼체로 통일**: `-습니다`, `-드립니다`, `-십시오`
  - 반말체(`-한다`, `-했다`)는 인용·내레이션 등 문맥에서 제한적으로만 사용
  - 같은 글 안에서 합쇼체와 반말체를 섞지 않는다
- **경전 직접 인용은 한국어 경전체 그대로 유지**: `-느니라`, `-도다`,
  `-하시니라`, `-하시매` 등. 본문 합쇼체와 자연스럽게 공존시키고
  인용을 합쇼체로 풀어 쓰지 않는다.
  - 예: 본문 "...라고 말씀하셨습니다" + 인용 "...그가 살아나셨느니라"
- **청중·독자 호칭은 "여러분"**, 1인칭 자기 지칭은 "저/제/저희"
- **기도문**: 합쇼체 권장 (`-드립니다`). 옛 경어체(`-하옵소서`)는 자연스러운
  현대 표현으로 풀어 쓴다

### 신성한 호칭·인칭대명사

영어는 He/Him/His 대문자로 신성을 표시하지만 한국어는 인칭대명사가 아니라
**호칭**으로 표현한다. 영어 인칭대명사를 "그/그를/그의"로 직역하지 않는다.

| 영어 (신성) | 권장 한국어 |
| --- | --- |
| He (= God / Christ) | 하나님 / 주님 / 구주 / 그분 |
| Him | 주님께 / 그분께 |
| His | 주님의 / 하나님의 / 그분의 |
| His Atonement | 주님의 속죄 (그분의 속죄도 가능) |

- 직역 회피 예: "He loves us" → "주님은 우리를 사랑하십니다" (○)
  / "그는 우리를 사랑한다" (×)
- 한 단락 안에서 호칭이 너무 반복되면 "그분"으로 적절히 변주

### 문장 구성 (영어 직역 회피)

- **수동태 → 능동태**: 한국어는 행위 주체를 드러내는 능동 구성이 자연스럽다.
  - "We are commanded to pray." → "주께서 우리에게 기도하라고 명하셨습니다."
- **추상명사 → 동사구**: 영어식 명사화를 풀어 쓴다.
  - "the application of faith" → "신앙을 발휘하는 것"
  - "the importance of repentance" → "회개가 얼마나 중요한지"
- **긴 관계절 → 분할**: 영어의 길게 이어지는 which/that 절은 한국어에서
  짧은 문장 두세 개로 끊는다.
- **연결어 자연화**: "and / but / so"를 매번 "그리고 / 그러나 / 그래서"로
  옮기지 않는다. 한국어는 종결어미·연결어미로 자연스럽게 잇는다.
- **한 문장에 절을 3개 이상 쌓지 않는다**: 호흡이 끊기면 분리한다.

### 단어 선택

- **신학적 한자어는 그대로 유지**: 속죄, 회개, 성신, 계명, 신권, 의식,
  성약 등 (1장 핵심 용어 참조)
- **그 외 일반 표현은 자연스러운 순한국어 선호**: 한자어 남발 회피
- **강조어 직역 금지**: very / really / truly / indeed 를 매번 "매우 /
  정말 / 진실로"로 옮기지 않는다. 한국어는 어조와 어순으로 강조하는 경우가
  많다. 굳이 옮긴다면 "참으로 / 실로 / 정녕".
- **영어식 추상명사 나열 자제**: "~의 가능성", "~의 중요성", "~의 필요성"
  같은 표현 남발 회피
- **구어체·신조어·외래어 남용 금지**: "진짜, 엄청, 되게, 너무" 같은 구어
  강조어, 불필요한 외래어, 인터넷 표현 피한다
- **같은 단어 반복 회피**: 한 단락에서 같은 어휘 반복보다 호응되는 다른
  표현으로 변주 (단, 신학 핵심 용어는 일관성 유지가 우선)

### 시제·서법

- **영어 현재완료** (has shown / have done) → 문맥에 따라 한국어 과거
  (`보이셨다`) 또는 현재 완료적 의미 (`보이셨고 지금도 그러하다`)로 옮긴다
- **명령형 권고**: 합쇼체 권고형 `-십시오` 또는 청유형 `-합시다`
- **가정법**: if + would → `-면 ... 할 것입니다` (would 자체는 한국어에서
  생략해도 자연스러우면 생략)

### 좋은 예 / 피해야 할 직역

| 영어 원문 (예시) | 자연스러운 한국어 | 어색한 직역 |
| --- | --- | --- |
| He loves us. | 주님은 우리를 사랑하십니다. | 그는 우리를 사랑한다. |
| We are commanded to pray. | 주께서 기도하라고 명하셨습니다. | 우리는 기도하도록 명령받습니다. |

## 6. 인용·따옴표·영문 병기 규칙

| 강조 (드물게) | 굵게 또는 작은따옴표 | **신권** 또는 '신권' |

### 영문 병기

- 처음 등장하는 인명·고유명사는 한글 표기 + 영문을 괄호로 병기할 수 있다.
  - 예: 게릿 더블유 공 장로(Gerrit W. Gong)
- 공식 한국어 표기가 잘 알려져 있고 문서 안에서 반복되면 두 번째부터는 한글만.
- 일반 인명(공식 표기 없는 사람) 처음 등장 시: 한글(영문) 형태 권장.
  - 예: 세라 윌리엄스(Sarah Williams)

### 약어·이니셜

- 영문 약어는 처음 등장 시 풀어 쓰는 것이 원칙. 약칭 병기는 선택.

- 중간 이니셜은 한글 음가로 표기하고 마침표 생략 (인명 참고집 2장 참조).

## 8. 경전 인용 처리 절차

경전 구절을 미리 정리해 두지 않는다. 경전 구절이 직접 인용되면 그 자리에서
임의로 번역하지 말고, 반드시 공식 한국어 경전 문구를 찾아 그대로 사용한다.

1. 인용된 구절의 책·장·절을 확인한다.
2. churchofjesuschrist.org 한국어판 경전에서 공식 한국어 문구를 찾는다.
   웹 검색을 활용한다. (검색 예: "교리와 성약 82편 3절 교회",
   "앨마서 37장 6절 churchofjesuschrist", site:churchofjesuschrist.org)
3. 공식 문구를 찾으면 그대로 사용한다. 원문이 구절의 일부만 인용한
   경우에는 공식 문구에서 해당 부분만 발췌하되, 어미·어순·표현을 임의로
   바꾸지 않는다.
4. 웹 검색으로도 공식 문구를 확인할 수 없을 때만 자연스러운 임시 번역을
   제공하고 `[공식 경전 문구 확인 필요]`로 표시한다.

## 9. 지도자 말씀 인용 처리 절차

지도자 말씀의 제목·인용문을 미리 정리해 두지 않는다. 지도자 말씀이
직접 인용되면 그 자리에서 임의로 번역하지 말고, 공식 한국어 문구를
웹 검색으로 찾아 그대로 사용한다. (경전 인용 처리 절차와 동일한 원칙)

1. 인용된 말씀의 화자, 제목, 모임(연차 대회·영적 모임 등)과 연도를
   확인한다.
2. churchofjesuschrist.org 한국어판에서 공식 한국어 자료를 찾는다.
   웹 검색을 활용한다.
   - 연차 대회 말씀: `study/general-conference` 한국어판
   - 영적 모임(devotional) 말씀: `study/broadcasts` 한국어판
   - 검색 예: 영어 제목 + 화자 + churchofjesuschrist + lang=kor
3. 찾은 페이지에서 다음을 그대로 가져온다.
   - **말씀 제목**: 공식 한국어 제목
   - **모임명**: 공식 한국어 명칭 (devotional → "영적 모임")
   - **인용문**: 공식 한국어 번역문. 원문이 일부만 인용했거나 삽입구를
     더한 경우, 공식 문구에서 해당 부분만 발췌하되 어미·어순·표현은
     임의로 바꾸지 않는다.
4. 웹 검색으로도 공식 문구를 확인할 수 없을 때만 자연스러운 임시
   번역을 제공하고 `[공식 번역 확인 필요]`로 표시한다.

> 한 번 검증한 말씀이라도 이 문서에 인용문 전체를 축적하지 않는다.
> 번역할 때마다 위 절차로 공식 문구를 확인한다. 단, 자주 인용되는
> 화자의 대표 말씀 **제목** 정도는 인명 참고집의 해당 인물 항목에
> 간단히 메모해 둘 수 있다 (예: 인명 참고집의 넬슨 회장·밸라드 장로·
> 공 장로 항목 참조).

## 10. 말씀·기도 맺음말 ("In the name of Jesus Christ, amen.")

영어의 맺음말 "In the name of Jesus Christ, amen."을 "예수 그리스도의
이름으로 아멘."으로 직역하면 한국어에서 서술어가 없어 어색하다. 한국어
교회 자료는 맥락에 맞는 동사를 넣어 맺는다.

| 맥락 | 권장 한국어 맺음말 |
| --- | --- |
| 말씀·설교를 맺을 때 (가장 일반적) | 예수 그리스도의 이름으로 말씀드립니다. 아멘. |
| 말씀·설교를 맺을 때 (전한다는 어감) | 예수 그리스도의 이름으로 전합니다. 아멘. |
| 마지막이 간증일 때 | 예수 그리스도의 이름으로 간증합니다. 아멘. |
| 기도를 맺을 때 | 예수 그리스도의 이름으로 기도드립니다. 아멘. |
| 회중에게 축복을 더할 때 | 예수 그리스도의 이름으로 간구하고 축복합니다. 아멘. |

- **맺음말 동사는 영어 원문의 동사가 아니라 본문의 성격으로 결정한다.**
  영어가 "I pray in the name of Jesus Christ"라도 본문이 말씀(설교)이면
  "기도드립니다"가 아니라 "말씀드립니다" 또는 "전합니다"를 쓴다.
- 원문이 "holy / sacred / sacred and holy" 등 수식어를 결합한 경우 그대로 살린다.
  - holy name → 거룩한 이름
  - sacred name → 성스러운 이름
  - sacred and holy name → 거룩하고 성스러운 이름
  - 예: "예수 그리스도의 거룩하고 성스러운 이름으로 전합니다. 아멘."
- 피해야 할 번역: "예수 그리스도의 이름으로 아멘." (서술어 누락)

## 11. 번역 후 점검 항목

### 정확성
1. 공식 교회 한국어 용어와 충돌하지 않는가
2. 교리적 의미가 약화·과장되지 않았는가
3. 인명·중간 이니셜이 교회 관례와 일치하는가 (인명 참고집 대조)
4. 직책 호칭(장로/회장/자매 등)이 원문 시점에 맞게 부여되었는가
5. 경전 인용과 장절 표기가 정확한가
6. 원문에 없는 해석·설명을 추가하지 않았는가

### 어체·문체 (5장)
7. 본문이 합쇼체로 일관되어 있는가 (반말체 섞이지 않음)
8. 경전 직접 인용은 한국어 경전체(`-느니라`, `-도다` 등) 그대로 유지되었는가
9. 신성한 인칭대명사(He/Him/His)를 "그/그를/그의"로 직역하지 않고
   호칭(주님/하나님/그분)으로 처리했는가
10. 영어 직역(수동태 남발·긴 관계절·강조어 직역·`의의의` 나열) 회피되었는가
11. 맺음말 동사가 본문 성격에 맞는가 (10장)

### 표기
12. 책·잡지·찬송가 제목(『 』)과 말씀·기사 제목(" ") 표기가 일관된가
13. 영문 약어·고유명사가 일관되게 표기되었는가 (6장)

### 미해결 항목
14. 불확실한 용어·인명·경전 문구는 `[확인 필요]`로 표시했는가
